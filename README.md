**SoloCore** 是一个开源项目，致力于帮助程序员、创造者和有执行力的人成为“AI时代的超级个体”，打造成功的“一人公司”。

在这里，我们将开源一个完整的产品级项目，并围绕它构建一套强大的AI智能体工具，同时分享“Vibe Coding”和个人IP打造的实战教程。

---

## 🎯 我们的使命 (The Mission)

在AI时代，个体的力量正在被空前放大。“一人公司”不再是遥不可及的梦想。

`SoloCore` 的使命是提供一套**开源的、可本地部署的AI工具栈**，并结合**透明的实战教程**，赋能每一个有志向的个体，使其能独立完成产品、营销和商业化的闭环。

## 💡 项目核心：从 TextLoom 开始

`SoloCore` 项目的核心是 `TextLoom`，这是一个“文章一键转短视频”的产品级项目。我们将在这里**逐步、完整地**将其开源。

但这仅仅是开始。我们将在此基础上不断迭代，集成更多的AI智能体，构建一个强大的内容创作与分发工作流。

## ✨ 核心功能 (Features)

* **[已规划] TextLoom (核心组件):**
    * **文章转视频:** 自动将你的博客文章、笔记或任何长文本转换成引人入胜的短视频。
    * **AI配音 & 字幕:** 自动生成自然的AI旁白和同步字幕。
    * **素材匹配:** 智能匹配免版权的视频素材和图片。

* **[未来] 写作智能体 (Writing Agents):**
    * **公众号智能体:** 帮助你构思、写作和排版深度图文。
    * **小红书智能体:** 自动生成高“种草”率的图文笔记。

* **[进行中] 实战教程 (Tutorials):**
    * **Vibe Coding:** 带你从0到1构建 `SoloCore`，展示产品级项目的思考与实现。
    * **个人IP打造:** 分享我是如何使用 `SoloCore` 工具栈打造个人IP，获取流量和用户的。

## 👤 谁适合这个项目？

`SoloCore` 为两类人群设计：

1.  **开发者 & 学习者:**
    * 你想学习一个**真实的产品级项目**是如何构建的，而不是玩具代码。
    * 你对AI智能体、内容创作工具的后端和前端实现感兴趣。
    * 你可以通过学习源码、贡献代码，来提升你的全栈和AI工程能力。

2.  **创作者 & “一人公司”实践者:**
    * 你不想（或没时间）深入代码，你只想**在本地运行这个产品**。
    * 你需要一个强大的自动化工具来帮你写文章、做视频，解放你的生产力。
    * 你可以直接使用 `SoloCore` 来为你的个人IP或业务赋能。

## 🌟 我的其他开源项目

### [Easegen - 数字人课程创建平台](https://github.com/taoofagi/easegen-front)

一个开源的数字人课程制作平台，提供从课程制作、视频管理到智能出题的全方位解决方案。

**🏆 获奖记录：** 2024 华彩杯算力大赛总决赛二等奖

**核心功能：**
- 🎬 **2D/3D 数字人视频合成** - 支持传统 2D 和高保真 3D 数字人方案
- 🤖 **AI 智能辅助** - 智能课件生成、AI 自动出题、文档解析、脚本优化
- 📚 **课程模板系统** - 快速创建专业课程内容
- 🎨 **数字人定制** - 支持形象、声音和演播室自定义
- 📊 **PPT 集成** - 智能解析和展示课件内容

**技术栈：**
- **前端:** Vue 3 + TypeScript + Element Plus + Vite → [easegen-front](https://github.com/taoofagi/easegen-front)
- **后端:** Java/Spring Boot (基于 Ruoyi-Vue-Pro) → [easegen-admin](https://github.com/taoofagi/easegen-admin)

**📦 本地访问：**

这两个项目已经作为 Git Submodule 添加到 SoloCore 根目录，你可以直接访问：
```bash
# 克隆 SoloCore 时包含子模块
git clone --recursive https://github.com/haojing8312/SoloCore.git

# 或者在已克隆的仓库中初始化子模块
git submodule update --init --recursive

# 子模块位置
./easegen-front/   # 前端项目
./easegen-admin/   # 后端项目
```
